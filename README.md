# SwingTrack - B3 Swing Trade

Aplicação web para gerenciamento de operações de Swing Trade na Bolsa de Valores Brasileira (B3).

## Tecnologias

- Backend: Node.js, Express, SQLite, Yahoo Finance API
- Frontend: React, Vite, Tailwind CSS, Recharts

## Como executar

### Backend

```bash
cd backend
npm install
npm run dev
